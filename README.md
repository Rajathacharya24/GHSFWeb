# Greenfield Public School Website

This is a modern static school website. It uses only HTML, CSS, JavaScript, and local image assets, so it is easy to host for free.

## Edit School Details

Open `index.html` and replace:

- `Greenfield Public School`
- Address and phone number
- Email address
- Notices and dates
- Admission text

## Free Hosting Options

### Option 1: Netlify

1. Go to https://www.netlify.com/
2. Create a free account.
3. Drag this whole folder into Netlify's deploy area.
4. Netlify will give you a free website link.

### Option 2: GitHub Pages

1. Create a GitHub account.
2. Create a new repository.
3. Upload these files.
4. Open repository settings, then Pages.
5. Choose the main branch and save.

### Option 3: Vercel

1. Go to https://vercel.com/
2. Create a free account.
3. Import the project folder.
4. Deploy it as a static site.

## Files

- `index.html` - website content
- `styles.css` - design and responsive layout
- `script.js` - mobile navigation
- `assets/school-photo.png` - homepage banner photo
- `assets/campus-hero.png` - backup generated banner image
