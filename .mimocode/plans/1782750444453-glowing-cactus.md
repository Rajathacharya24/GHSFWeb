# Animation Plan for GHSFWeb

## Overview
Add tasteful, performant CSS animations to the school website. All animations go in `premium.css` (empty placeholder, already linked in `index.html`). A small JS addition in `script.js` will trigger scroll-based reveals via Intersection Observer.

## Files to Modify
- **`premium.css`** — all animation CSS (keyframes, transitions, reveal states)
- **`script.js`** — add scroll-triggered reveal observer + stagger logic

## Animations

### 1. Page Load — Hero Section
- Hero content fades in + slides up from 30px below over 0.8s ease-out
- Quick-links bar slides up with a 0.3s delay

### 2. Scroll Reveals — All Sections
- `.section`, `.stats-band`, `.notice-list article`, `.life-grid div`, `.admission-steps div` get a `.reveal` class via JS IntersectionObserver
- Reveal: opacity 0→1 + translateY(30px→0) over 0.6s ease-out
- Staggered children: each child delays by 0.1s (using `--i` CSS variable set by JS)

### 3. Hover Effects
- Buttons: scale(1.03) + shadow lift on hover, 0.2s transition
- Quick-links cards: subtle background tint + translateY(-2px) on hover
- Life-grid items: border-left color intensifies, slight scale on hover
- Nav links: already have background hover, add 0.2s transition

### 4. Mobile Nav
- Smooth slide-down with max-height transition + opacity fade (0→1)

### 5. Stats Band
- Fade-in on scroll (already has JS counter, just add the reveal class)

## Implementation Detail

### premium.css additions:
```css
/* === KEYFRAMES === */
@keyframes fadeSlideUp {
  from { opacity: 0; transform: translateY(30px); }
  to   { opacity: 1; transform: translateY(0); }
}

@keyframes heroReveal {
  from { opacity: 0; transform: translateY(40px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* === HERO LOAD ANIMATION === */
.hero-content { animation: heroReveal 0.8s ease-out 0.2s both; }
.quick-links  { animation: heroReveal 0.6s ease-out 0.5s both; }

/* === SCROLL REVEAL === */
.reveal {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.6s ease-out, transform 0.6s ease-out;
  transition-delay: calc(var(--i, 0) * 0.1s);
}
.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}

/* === HOVER EFFECTS === */
.button { transition: transform 0.2s ease, box-shadow 0.2s ease; }
.button:hover { transform: scale(1.03); box-shadow: 0 6px 20px rgba(0,0,0,0.15); }
.quick-links a { transition: background 0.2s ease, transform 0.2s ease; }
.quick-links a:hover { background: #f0f5fb; transform: translateY(-2px); }
.life-grid div { transition: border-color 0.2s ease, transform 0.2s ease; }
.life-grid div:hover { border-color: #1a5c44; transform: scale(1.02); }
.site-nav a { transition: color 0.2s ease, background 0.2s ease; }
.notice-list article { transition: transform 0.2s ease, box-shadow 0.2s ease; }
.notice-list article:hover { transform: translateY(-3px); box-shadow: 0 8px 24px rgba(0,0,0,0.08); }
```

### script.js additions:
```js
// Scroll-triggered reveal with stagger
const revealEls = document.querySelectorAll(
  '.section, .stats-band, .notice-list article, .life-grid div, .admission-steps div'
);
revealEls.forEach((el, i) => {
  el.classList.add('reveal');
  // For staggered children, set --i on each child
  if (el.children.length) {
    [...el.children].forEach((child, j) => {
      child.classList.add('reveal');
      child.style.setProperty('--i', j);
    });
  }
});

const revealObserver = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
      revealObserver.unobserve(entry.target);
    }
  });
}, { threshold: 0.15 });

document.querySelectorAll('.reveal').forEach(el => revealObserver.observe(el));
```
